# Earthquake Tracker

This PHP-CLI application is used to fetch the latest earthquakes from the Kandilli Observatory's website.

## Usage

1. Make sure the file is executable:

    ```bash
    chmod +x deprem.php
    ```

2. Run the application:

    ```bash
    ./deprem.php
    ```

## Requirements

- PHP must be installed.
